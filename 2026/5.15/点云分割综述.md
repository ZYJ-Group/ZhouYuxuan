# 点云分割的综述阅读

## 1.背景与问题概述

- 背景：在自动驾驶场景下，相比3D目标检测，点云语义分割可以给点云中的每一个点都分配类别。但3D数据在内存消耗和推理时间方面的复杂性使得任务具有挑战性。
- 问题：怎样高效、准确地对点云进行语义分割

## 2.主要内容

###  基础概念

- 稀疏卷积与子流形稀疏卷积

- 自注意机制

- 生成对抗网络

- 扩张的卷积

- 早期点云语义分割方法通常被划分为**投影方法、体素方法和点方法**，最近的方法通常结合了不同的表示

  | 方法类别                      | 关键词             | 主要优势               | 主要问题                     |
  | ----------------------------- | ------------------ | ---------------------- | ---------------------------- |
  | **Projection-based 投影方法** | 3D 点云转 2D 图像  | 速度快、计算量小       | 丢失三维几何信息             |
  | **Voxel-based 体素方法**      | 点云划分为 3D 网格 | 保留空间结构、精度较高 | 显存和计算量大，小目标易丢失 |
  | **Point-based 点方法**        | 直接处理原始点     | 保留几何细节           | 邻域搜索慢，实时性差         |

###  点云语义分割

- 该论文把点云语义分割方法大致分成三类：**投影方法、三维方法和混合方法**。

| 方法类别                      | 核心思想                                                 | 具体形式以及代表方法                                         | 优点                                             | 缺点                                                   | 定位                                   |
| ----------------------------- | -------------------------------------------------------- | ------------------------------------------------------------ | ------------------------------------------------ | ------------------------------------------------------ | -------------------------------------- |
| **Projection-based methods ** | 将 3D LiDAR 点云投影到 2D 平面，再用 2D 图像分割网络处理 | **BEV 鸟瞰图投影**：PolarNet；**Range View 距离图投影**：SalsaNext | 计算量小，推理速度快，适合实时场景               | 3D 点云投影成 2D 会丢失空间几何信息，分割精度相对较低  | 早期和实时性任务中常用，主要优势是效率 |
| **3D-based methods **         | 尽量保留点云的三维空间结构，直接在 3D 空间中提取特征     | **体素方法**：Cylinder3D；**点-体素结合方法**：SPVC/SPVNAS、DRINet++、AF2-S3Net、JS3C-Net | 能保留更多 3D 几何信息，分割精度通常高于投影方法 | 计算复杂度高，显存消耗大；体素分辨率低时容易丢失小目标 | 当前主流高精度方法的重要类别           |
| **Hybrid methods **           | 融合 2D 投影特征和 3D 点云特征，兼顾速度和精度           | AMVNet：融合 RV 和 BEV；2D3DNet：用 2D 分割监督 3D 模型；RPVNet：融合 Range、Point、Voxel 三种表示 | 能综合不同表示方式的优点，通常表现更全面         | 网络结构更复杂，特征融合设计难度更高                   | 近年来的重要发展趋势                   |

## 3.数据集与评价指标

​	论文还介绍了一些常用数据集，比如 SemanticKITTI、nuScenes-Lidarseg 和 SynLiDAR。

​	SemanticKITTI 和 nuScenes 都是真实自动驾驶数据集，常用来测试模型效果。SynLiDAR 是合成数据集，可以缓解真实点云标注困难的问题。

​	评价指标主要是 mIoU，也就是平均交并比，用来衡量模型预测出来的类别和真实类别重合得好不好。

## 4.总结

​	读完这篇论文后，我觉得点云语义分割的核心问题就是：怎样在精度和速度之间取得平衡。

​	如果只追求速度，可能会损失三维信息；如果只追求精度，模型又可能太复杂，不能实时运行。所以实际应用中不能只看 mIoU，还要看推理时间、显存消耗和硬件条件。

​	这篇综述最大的作用是帮我建立了一个整体框架。以后再看具体算法论文时，我可以先判断它属于哪一类方法，它解决的主要问题是什么，它的优缺点在哪里
